# sparrowhawk
<p align="center"><img src="_img/keymap.png" width="1000"></p>

sparrowhawk (sphk) is a steno theory built for computer users. not court reporters.

conventional steno theories were designed for speech transcription in the 1950s. sphk adds new keys, new theory, and enables the following as first-class citizens:

- standard 200+ WPM stenographic input of natural languages (english/japanese)
- text navigation/selection/editing from the home row, everywhere
- semantic symbol and unicode character input
- chorded tenkey numpad

nine years of daily use to date. actively updated.

## demo
song lyric steno demo. bootstrapped on ErgoDox EZ, now on purpose-built [aodox](https://github.com/jak-crow/aodox).
